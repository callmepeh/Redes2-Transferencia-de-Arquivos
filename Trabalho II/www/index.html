Sobre o Projeto

Este é o Mini-Servidor Web desenvolvido para a Terceira Avaliação de Redes de Computadores II.

O sistema evolui o trabalho anterior de transferência de arquivos (TCP vs R-UDP) para um servidor HTTP/1.1 funcional, com:

  • ✅ Resolução de Nomes via Mini-DNS (UDP)
  • ✅ Servidor HTTP/1.1 sobre TCP nativo
  • ✅ Servidor HTTP/1.1 sobre R-UDP (Reliable UDP)
  • ✅ Cabeçalho personalizado X-Custom-Auth
  • ✅ Simulação de rede com tc/netem (perda e latência)
  • ✅ Validação com Wireshark/TCPDump
